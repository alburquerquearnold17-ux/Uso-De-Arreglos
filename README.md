# Uso-De-Arreglos
Este programa lo que hace es guardar las notas de hasta 20 estudiantes y saca el promedio de cada uno y del grupo entero y busca la nota más alta y la más bajita, y al final te dice quiénes están pasan y quiénes se quemaron todo usando arreglos y bucles para que quede organizado y fácil de manejar.
