# Uso-De-Arreglos
Este programa lo que hace es guardar las notas de 20 estudiantes y saca el promedio de cada uno y del grupo entero y busca la nota más alta y la más bajita, y al final te dice quiénes pasan y quiénes se quemaron todo usando arreglos y bucles para que quede organizado y fácil de manejar.

Explicacion breve de como funcionan los arreglos en la solucion de mi programa:

Los arreglos de mi programa son como una caja para guardar las notas de los estudiantes donde cada posicion es una nota de un estudiate con bucles asi el programa recorre esas posiciones para sacar el promedio y decir la nota mas bajita y la mas alta.
